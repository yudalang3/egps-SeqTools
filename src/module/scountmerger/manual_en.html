  1. Check required parameters in the parameter area (switch examples if provided)

  4. After completion, verify output files in the output directory

2. Overview

The following section contains the module-specific description and usage notes (including any existing notes/external links).

Task to resolve

The count matrix files produced by the `featureCounts` software is read and then a file is output with the expression matrix.

Quick start

All user actions are straightforward. Please follow the GUI instructions.

  1. View the required input data in the parameter input area. When the module provides more than one example, you can click the example button to see the next.
  2. The user enters the required parameters.
  3. Click the Execute button to see the results.

3. Inputs & Outputs (General)

Item Description
Inputs Typically text/table/sequence files (FASTA/TSV/CSV/GFF3, etc.) or URL/directory paths. Follow the GUI parameter hints.
Outputs Typically result files (TSV/FASTA/PNG/PDF, etc.) or console text. Using a dedicated output directory is recommended.
Encoding UTF-8 is recommended. If you see garbled text, check file encoding and system locale.

4. Parameter & Run Tips

  • Paths with spaces: prefer directories without spaces; quote paths if the module supports it.
  • Large files: validate with a small sample first, then run the full dataset.
  • Result check: verify row/entry counts and key fields to ensure the run is correct.

5. FAQ

Symptom Possible Cause Suggestion
No output / very slow Large input, slow network, wrong parameters Try a small sample first; ensure output directory is writable; check logs
File not found Wrong path / permission / file locked Use absolute paths; check permissions; close programs that hold the file
Empty results Filters too strict / wrong column index / wrong delimiter Relax filters; confirm input format; verify the selected column
